# Copilot : Quick References

## Prompt structure

```text
Context: relevant project or document
Task: one explicit objective
Constraints: format, scope, exclusions
Sources: files or data to use
Output: exact expected structure
```

## Useful prompt pattern

```text
Using [source], produce [deliverable] for [audience].
Include [required elements].
Exclude [out-of-scope elements].
Return the result as [format].
```

## Review checklist

```text
Verify factual claims
Check links and commands
Remove unsupported assumptions
Confirm sensitive data handling
Test generated code before committing
```
