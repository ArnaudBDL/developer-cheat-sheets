# Copilot : Overview

## Product Families

```text
Microsoft 365 Copilot     Work chat, Microsoft 365 apps and organizational context
Microsoft Copilot Studio  Build, extend, test, publish and govern agents
GitHub Copilot            AI assistance for software development workflows
Security Copilot          AI assistance and agents for security operations
```

## Core Concepts

```text
Prompt      Instruction or question from the user
Context     Information available for grounding the response
Agent       Specialized assistant with instructions, knowledge and tools
Action      Operation performed through an approved tool or workflow
Connector   Integration that exposes external information or capabilities
Grounding   Use of selected sources to support a response
```

## Working Pattern

```text
1. Define the outcome.
2. Provide relevant context and sources.
3. State format and constraints.
4. Review citations and generated content.
5. Refine the prompt or correct the source context.
6. Keep human approval for consequential actions.
```
