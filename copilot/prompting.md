# Copilot : Prompting

## Prompt Structure

```text
Goal          What Copilot should accomplish
Context       Audience, background and business situation
Expectations  Format, tone, length, constraints and quality criteria
Source        Files, messages, meetings, repositories or connected systems to use
```

## Template

```text
Create [deliverable] for [audience].
Use [source or current content].
Include [required elements].
Exclude [unwanted elements].
Return the result as [format].
```

## Refinement

```text
Make the result shorter and preserve all decisions.
Separate confirmed facts from assumptions.
Add citations for every factual claim.
Rewrite for a technical audience.
List missing information without inventing it.
```

## Rules

- Give one clear primary objective.
- Name the source, scope and time period explicitly.
- Define the expected output structure.
- Provide examples only when they clarify a pattern.
- Review and verify important results before use.
