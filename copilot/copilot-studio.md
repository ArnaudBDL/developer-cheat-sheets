## Release Checklist

- Use separate environments for development, test and production.
- Configure authentication and data policies.
- Review knowledge permissions and sensitivity.
- Test tools with least-privileged identities.
- Run functional, safety and regression evaluations.
- Review warnings before publication.
- Enable audit, monitoring and operational ownership.
