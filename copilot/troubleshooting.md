# Copilot : Troubleshooting

## Poor or Incomplete Response

```text
Clarify the goal.
Name the exact source and time range.
Reduce irrelevant context.
Specify the output format.
Ask for citations.
Break a compound request into focused steps.
```

## Missing Work or Connector Content

```text
Confirm the user can open the source item directly.
Check that the connector is enabled for the organization.
Use the exact project, file or system name.
Narrow the query and provide a date range.
Allow for connector indexing or synchronization.
Review source permissions and deletions.
```

## Agent or Action Failure

```text
Confirm the agent is available in the current surface.
Check authentication and environment access.
Review tool permissions and data policies.
Validate required parameters.
Inspect run, session and audit logs.
Test the tool independently with the same identity.
```

## GitHub Copilot Agent

```text
Confirm the paid plan and repository availability.
Check organization and repository policies.
Review the agent session log.
Inspect branch protections and required checks.
Run tests and review the draft pull request before merge.
```

## Escalation Evidence

```text
Product and surface
Tenant, environment or repository
User and agent identifier
Exact prompt
Expected and actual result
Timestamp and correlation identifier
Source or connector involved
Relevant logs and screenshots
```
