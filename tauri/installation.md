# Tauri : Installation

## Prerequisites

```bash
rustc --version
cargo --version
node --version
npm --version
```

## Create a Project

```bash
npm create tauri-app@latest
cd application
npm install
npm run tauri dev
```

## Add to an Existing Frontend

```bash
npm install --save-dev @tauri-apps/cli@latest
npm run tauri init
npm run tauri info
```
