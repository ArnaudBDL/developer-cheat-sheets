# Networking : Security

## Core Controls

```text
Network segmentation
Least-privilege firewall rules
Strong identity and authentication
Encryption in transit
Secure administrative access
Controlled ingress and egress
Patch and vulnerability management
Centralized logging and monitoring
Backups of network configuration
Incident response procedures
```

## Threats

```text
Eavesdropping
Address or route spoofing
DNS manipulation
Unauthorized lateral movement
Credential theft
Denial of service
Exposed management services
Misconfigured cloud security rules
Rogue wireless access
Data exfiltration
```

## Hardening Checklist

- Separate user, server, management and guest networks.
- Restrict administrative protocols to approved sources.
- Disable unused services and legacy protocols.
- Use authenticated and encrypted alternatives.
- Protect routing, DNS, DHCP and time sources.
- Monitor configuration drift and unexpected listening ports.
- Rotate credentials and certificates.
- Test restore and failover procedures.
- Preserve logs with synchronized timestamps.
- Document accepted risks and rule ownership.
