# Java : Syntax

## Program

```java
package com.example;

public final class Main {
    public static void main(String[] args) {
        String name = "Java";
        System.out.printf("Hello %s%n", name);
    }
}
```

Java is case-sensitive. Statements normally end with semicolons, braces delimit blocks, and source files conventionally use UTF-8. Match public top-level class names to filenames and keep one clear responsibility per type.
