# Node.js : NPM

## Project Commands

```bash
npm init -y
npm install
npm ci
npm install PACKAGE
npm install --save-dev PACKAGE
npm uninstall PACKAGE
npm update
```

## Run Scripts

```bash
npm run
npm run build
npm test
npm start
npm exec COMMAND
npx COMMAND
```

## Inspect Dependencies

```bash
npm list
npm list --depth=0
npm outdated
npm explain PACKAGE
npm view PACKAGE
npm audit
```

## Configuration

```bash
npm config list
npm config get registry
npm config set fund false
npm cache verify
```

## Global Packages

```bash
npm install --global PACKAGE
npm list --global --depth=0
npm uninstall --global PACKAGE
```
