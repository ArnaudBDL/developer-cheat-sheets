# .NET : CLI

## Command Structure

```text
dotnet <command> [arguments] [options]
```

## Core Commands

```bash
dotnet new list
dotnet restore
dotnet build
dotnet run
dotnet test
dotnet publish
dotnet clean
dotnet format
```

## Help

```bash
dotnet --help
dotnet build --help
```

The CLI can execute SDK commands or run a framework-dependent DLL. Run commands from the intended project or solution directory and make configuration and framework selections explicit in automation.
