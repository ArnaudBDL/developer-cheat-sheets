# Python : Syntax

## Basic Program

```python
name = "Arnaud"
count = 3

if count > 0:
    print(f"Hello {name}: {count}")
```

## Rules

```text
Indentation defines blocks.
Names are case-sensitive.
Lines normally end without semicolons.
Comments begin with #.
Docstrings use string literals as the first statement of modules, classes or functions.
```

Use four spaces per indentation level, UTF-8 source files, descriptive identifiers, and automated formatting and linting agreed by the project.
