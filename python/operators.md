# Python : Operators

## Operators

```python
+ - * / // % **
== != < <= > >=
and or not
in not in
is is not
& | ^ ~ << >>
:=
```

## Examples

```python
quotient = 17 // 5
remainder = 17 % 5
same_object = first is second
equal_value = first == second
```

Use `is` for identity checks, especially `value is None`, and `==` for value equality. Parenthesize mixed expressions when precedence is not immediately obvious.
