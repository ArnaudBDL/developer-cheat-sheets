# Python : Standard Library

## Common Modules

```text
pathlib, os, shutil, tempfile
json, csv, sqlite3
collections, itertools, functools
math, decimal, statistics
re, datetime, zoneinfo
logging, argparse, configparser
subprocess, threading, multiprocessing, asyncio
hashlib, hmac, secrets
unittest, doctest
```

Prefer the standard library when it meets the requirement, but verify security, performance and protocol needs. Read the exact Python-version documentation because APIs and behavior can change between releases.
