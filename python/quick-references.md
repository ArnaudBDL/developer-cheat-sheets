# Python : Quick References

## Environment

```bash
python --version
python -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
python -m pip check
```

## Project Checks

```bash
python -m unittest
python -m compileall src
python -m application
```

## Debugging

```bash
python -X dev -m application
python -m pdb script.py
python -m cProfile -m application
```
