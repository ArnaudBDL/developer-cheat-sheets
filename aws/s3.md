# AWS : S3

## Buckets and Objects

```bash
aws s3 mb s3://UNIQUE_BUCKET_NAME --region eu-west-3
aws s3 ls
aws s3 ls s3://UNIQUE_BUCKET_NAME --recursive --human-readable
aws s3 cp document.txt s3://UNIQUE_BUCKET_NAME/documents/document.txt
aws s3 cp s3://UNIQUE_BUCKET_NAME/documents/document.txt .
aws s3 sync ./dist s3://UNIQUE_BUCKET_NAME/site --delete
aws s3 rm s3://UNIQUE_BUCKET_NAME/documents/document.txt
```

## Security and Versioning

```bash
aws s3api put-public-access-block   --bucket UNIQUE_BUCKET_NAME   --public-access-block-configuration   BlockPublicAcls=true,IgnorePublicAcls=true,BlockPublicPolicy=true,RestrictPublicBuckets=true

aws s3api put-bucket-versioning   --bucket UNIQUE_BUCKET_NAME   --versioning-configuration Status=Enabled

aws s3api get-bucket-encryption --bucket UNIQUE_BUCKET_NAME
```

Keep public access blocked unless explicitly required and use lifecycle policies for retention and storage-class transitions.
