# OWASP : Testing

## Testing Layers

```text
Threat modelling
Architecture review
Source-code review
Static analysis
Dependency analysis
Secret scanning
Infrastructure and configuration scanning
Dynamic application testing
API contract and authorization testing
Manual security testing
Retesting after remediation
```

## Test Scenarios

- Verify anonymous, authenticated and privileged behavior separately.
- Test horizontal, vertical and cross-tenant authorization boundaries.
- Validate malformed, missing, oversized and unexpected inputs.
- Verify session creation, rotation, expiration and revocation.
- Test application behavior when dependencies time out or fail.
- Verify logs and alerts for security-relevant events.
- Confirm secure defaults in a production-equivalent configuration.
- Test restore, rollback and incident procedures.
