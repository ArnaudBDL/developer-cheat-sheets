# Retrieval-Augmented Generation : Data Sources

## Source Types

```text
Documents and wikis
Object storage
Relational and document databases
Ticketing and CRM systems
Source repositories
Approved websites and APIs
Messages and transcripts where policy permits
```

## Required Metadata

```text
Source identifier
Canonical URI or location
Title and content type
Owner
Created and modified timestamps
Version
Language
Tenant and access rules
Retention state
```

- Prefer authoritative systems of record.
- Define inclusion, exclusion and deletion policies.
- Preserve source permissions and ownership.
- Detect duplicates and superseded versions.
- Record extraction failures and unsupported formats.
- Do not ingest secrets or unnecessary personal data.
