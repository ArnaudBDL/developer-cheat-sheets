# Git : Installation

## Install Git

```bash
# macOS
brew install git

# Debian / Ubuntu
sudo apt update
sudo apt install git

# Fedora
sudo dnf install git

# Arch Linux
sudo pacman -S git
```

## Verify Installation

```bash
git --version
git help
git help clone
git help --all
```

## Locate Git

```bash
command -v git
which git
git --exec-path
```

## Update Git

```bash
brew upgrade git
sudo apt update && sudo apt upgrade git
sudo dnf upgrade git
```
