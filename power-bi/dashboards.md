# Power BI : Dashboards

## Dashboard Characteristics

```text
Created in the Power BI service
Single canvas of pinned tiles
Can combine tiles from supported reports and semantic models
Designed for monitoring and navigation
Distinct from multi-page interactive reports
```

## Design Checks

- Pin only metrics needed for monitoring.
- Use titles that include context, period and unit.
- Keep related metrics together.
- Verify the destination report and filter behavior for each tile.
- Review alerts and refresh dependencies.
- Avoid treating dashboards as a replacement for detailed analytical reports.
- Confirm viewer permissions for every underlying item.
