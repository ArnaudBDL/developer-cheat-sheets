# Power BI : Visualizations

## Selection

```text
Comparison: bar or column chart
Trend: line chart
Composition: stacked chart where readable
Distribution: histogram or box-style custom visual where approved
Detail: table or matrix
KPI: card with context and target
Geography: map only with reliable geographic fields
```

## Design Checks

```text
Clear title and units
Accessible color contrast
Meaningful sort order
Readable labels
Consistent number formats
Controlled interactions
Useful alternative text
Keyboard and screen-reader considerations
```

Use the smallest set of visuals that answers the page objective. Avoid decorative clutter, misleading axes, excessive categories and unapproved custom visuals. Test cross-filtering, drill interactions and mobile layouts.
