# Power BI : Power BI Service

## Core Items

```text
Semantic models
Reports
Dashboards
Apps
Dataflows
Paginated reports
Workspaces
```

## Typical Flow

```text
Publish content to a workspace
Configure credentials and gateway mapping
Configure refresh where required
Assign workspace roles
Validate report and model permissions
Distribute through sharing or an app
Monitor usage, refresh and failures
```

The service hosts published analytics content and provides workspaces for collaboration. Distinguish content creation permissions from content consumption permissions, and verify semantic-model access separately from report visibility.
