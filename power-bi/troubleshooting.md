# Power BI : Troubleshooting

## Connection and Refresh

```text
Validate source availability and credentials.
Check privacy levels and parameters.
Confirm gateway mapping, version and drivers.
Inspect refresh history and gateway logs.
Verify schema changes and storage mode.
```

## Model and DAX

```text
Check relationship direction and cardinality.
Test measures in simplified filter contexts.
Verify date-table configuration.
Inspect totals separately from detail rows.
Check RLS role filters.
```

## Reports and Service

```text
Test the report in Desktop and the service.
Check workspace, report and semantic-model permissions.
Verify filters, bookmarks and interactions.
Inspect Performance Analyzer and capacity metrics where available.
Re-publish only after identifying the failing layer.
```

Record the workspace, item, source, timestamp, refresh identifier and exact error without copying credentials or sensitive data into support records.
