# Power BI : Power BI Desktop

## Authoring Views

```text
Report view   Builds pages and visual interactions
Data view     Inspects loaded tables and calculated objects
Model view    Manages tables, relationships and model metadata
Power Query Editor Transforms and loads source data
```

## Workflow

```text
Connect to sources
Transform data
Design the semantic model
Create measures
Build and test reports
Validate performance and security
Publish to a workspace
```

## Files

```text
.pbix   Combined Desktop report and semantic model file
.pbit   Template file without imported model data
.pbip   Power BI project format for source-control-oriented workflows
```

Keep source credentials out of shared files, define parameters for environment-specific values, and validate the report against the target service environment before publishing.
