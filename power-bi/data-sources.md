# Power BI : Data Sources

## Source Categories

```text
Files: Excel, CSV, JSON, XML, PDF and folders
Databases: SQL Server, PostgreSQL, MySQL, Oracle and cloud databases
Online services: SharePoint, Dataverse, Salesforce and supported SaaS systems
Interfaces: Web, OData, ODBC, APIs, R scripts and Python scripts
Microsoft Fabric and analytical platforms
```

## Connectivity Modes

```text
Import
DirectQuery
Direct Lake
Live connection
Composite model
```

## Checks

- Verify connector capabilities and authentication methods.
- Choose a storage mode based on freshness, scale, latency and source constraints.
- Apply privacy levels and data-source permissions correctly.
- Confirm whether a gateway is required after publishing.
- Avoid embedding credentials in queries or parameters.
- Test schema changes and source availability.
