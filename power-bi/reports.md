# Power BI : Reports

## Report Features

```text
Pages
Visual-level, page-level and report-level filters
Slicers
Drill down and drillthrough
Bookmarks
Buttons and page navigation
Report page tooltips
Conditional formatting
Mobile layout
```

## Page Pattern

```text
Title and active context
Summary KPIs
Primary analytical visual
Supporting breakdown
Filters and navigation
Definitions or notes where necessary
```

Keep each page focused on a user question. Use consistent navigation, preserve filter expectations, document metric definitions, test export behavior, and verify that every visual respects security and model relationships.
