# Power BI : Installation

## Products

```text
Power BI Desktop      Windows authoring application
Power BI service      Cloud service for publishing and collaboration
Power BI Mobile       Report consumption on supported mobile platforms
Power BI Report Builder Authoring paginated reports
Power BI Report Server On-premises report hosting
On-premises data gateway Connectivity between cloud services and local sources
```

## Checks

```text
Confirm operating-system support.
Install from the Microsoft Store or approved installer channel.
Verify tenant and licensing requirements for service features.
Keep Desktop and gateway versions within organizational support policy.
Install required database drivers only from trusted sources.
```

Power BI Desktop is the primary Windows application for creating interactive reports and semantic models. Separate installation and update procedures apply to Report Builder, Report Server, mobile clients, and the gateway.
