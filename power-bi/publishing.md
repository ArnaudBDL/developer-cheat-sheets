# Power BI : Publishing and Sharing

## Publishing Flow

```text
Select target workspace
Publish report and semantic model
Configure credentials and gateway mapping
Configure refresh
Validate RLS and workspace permissions
Review labels, descriptions and owners
Distribute through sharing or an app
```

## Distribution Options

```text
Workspace access
Direct sharing
Power BI app
Embedding where licensed and approved
Subscriptions
Export where policy permits
```

Do not use publishing as the security boundary. Validate access to the report, semantic model, workspace and source data. Prefer apps for governed audience distribution and maintain a clear content owner and support contact.
