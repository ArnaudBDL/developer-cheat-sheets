# Copilot Studio : Solutions and ALM

## ALM Components

```text
Power Platform solutions
Custom publisher
Environment variables
Connection references
Managed and unmanaged solution strategy
Source control
Deployment pipelines
Post-deployment configuration
```

## Lifecycle

```text
Develop -> Export or commit -> Validate -> Test -> Approve -> Deploy -> Configure -> Verify
```

Keep agents and dependencies in solutions, externalize environment-specific configuration, track changes in source control where supported, and avoid manual production edits that bypass the release process.
