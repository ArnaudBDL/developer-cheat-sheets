# Copilot Studio : Overview

## Purpose

```text
Copilot Studio is a graphical low-code platform for building agents and agent flows.
Agents can answer questions, use knowledge, call tools, react to triggers and run across supported channels.
```

## Core Building Blocks

```text
Instructions
Knowledge sources
Topics
Entities and variables
Tools, connectors and flows
Triggers
Authentication
Channels
Analytics and evaluation
```

## Lifecycle

```text
Plan -> Build -> Test -> Publish -> Monitor -> Improve
```

Keep each agent purpose-built, define measurable outcomes before building, and treat governance, security, testing and operations as part of the product rather than post-publication tasks.
