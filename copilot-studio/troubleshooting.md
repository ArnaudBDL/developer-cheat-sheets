# Copilot Studio : Troubleshooting

## Agent Behavior

```text
Check active instructions and orchestration mode.
Inspect topic triggers and overlapping intent.
Verify variables, entities and conversation context.
Reproduce with a minimal test utterance.
```

## Knowledge and Tools

```text
Confirm source availability and permissions.
Check indexing or source freshness.
Verify connector connection and tool schema.
Test the tool independently with the same identity.
Inspect timeout, rejection and error outputs.
```

## Deployment

```text
Confirm environment, solution version and published version.
Check authentication and channel configuration.
Validate environment variables and connection references.
Review data policies and sharing restrictions.
Inspect analytics, transcripts and approved telemetry.
```

Record the agent, environment, channel, identity type, timestamp, correlation evidence and exact outcome. Do not include secrets, tokens or unnecessary conversation content in support records.
