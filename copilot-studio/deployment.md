# Copilot Studio : Deployment

## Release Checklist

```text
Approved agent version
Environment variables
Connection references
Authentication
Data policies
Knowledge access
Tool credentials
Channel settings
Test results
Owners and support contacts
Rollback artifact
Monitoring
```

## Post-Deployment

```text
Verify agent availability
Test representative identities
Run knowledge and tool smoke tests
Confirm channel rendering
Confirm analytics and alerts
Validate sharing and audience
```

Move agents through development, test and production stages using controlled deployment. Publishing and deploying are lifecycle operations, not substitutes for authorization, governance or acceptance testing.
