# Copilot Studio : Security

## Security Layers

```text
Environment access
Maker and administrator roles
Authentication and identity
Tool authorization
Knowledge permissions
Endpoint security
Secret management
Data policies
Sharing and channel access
Audit logs
```

## Threats

```text
Prompt injection
Data exfiltration
Overprivileged connections
Cross-user or cross-tenant leakage
Unsafe autonomous actions
Malicious knowledge content
Unapproved publishing
Sensitive transcript exposure
```

Treat prompts and retrieved content as untrusted. Enforce access in source systems, use least privilege, require approvals for consequential actions, isolate environments, and regularly test authorization boundaries and data leakage scenarios.
