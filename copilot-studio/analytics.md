# Copilot Studio : Analytics

## Operational Signals

```text
Usage
Conversation outcomes
Topic performance
Knowledge gaps
Tool failures
Latency
Feedback
Escalation
Autonomous-agent health where applicable
```

## Improvement Loop

```text
Collect approved telemetry
Review failures and transcripts under policy
Classify the failing component
Update instructions, knowledge, topic or tool
Run regression evaluations
Deploy through the governed lifecycle
Monitor again
```

Define KPIs before launch. Protect transcript privacy, restrict analytics access, retain only what policy permits, and avoid interpreting a single metric without the underlying user outcome and failure context.
