# Copilot Studio : Agents

## Agent Composition

```text
Name and description
Instructions
Knowledge
Topics
Tools
Triggers
Conversation context
Authentication
Channel configuration
```

## Planning

```text
Audience
Business outcome
Supported tasks
Forbidden tasks
Authoritative sources
Required integrations
Escalation path
Success measures
Owner and support model
```

Create an agent from a clear scope rather than a broad aspiration. Define what it should do, what it must not do, when it should ask for clarification, and when it should stop or transfer control.
