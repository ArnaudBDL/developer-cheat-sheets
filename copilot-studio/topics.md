# Copilot Studio : Topics

## Topic Structure

```text
Name
Description
Trigger
Conversation nodes
Conditions
Questions and inputs
Messages
Actions
Topic transitions
End behavior
```

## Use Topics When

```text
The conversation needs predictable control.
Required information must be collected.
Business rules determine a path.
A regulated response must be curated.
A tool needs validated inputs.
```

Keep topics modular and focused. Use clear trigger intent, avoid overlapping topics, validate collected inputs, and define interruption, cancellation, retry and escalation behavior explicitly.
