# Copilot Studio : Generative AI

## Generative Capabilities

```text
Generate grounded answers from knowledge
Rewrite queries for retrieval
Summarize retrieved content
Select tools and plan actions through generative orchestration
Create or modify agent content from natural-language descriptions
```

## Controls

```text
Clear instructions
Approved knowledge
Tool allowlist
Authorization outside the model
Confirmation for consequential actions
Evaluation sets
Fallback and escalation
Monitoring
```

Generated output is probabilistic and can be incomplete or incorrect. Require evidence for factual claims, validate tool arguments, expose uncertainty, and keep human review for sensitive or consequential workflows.
