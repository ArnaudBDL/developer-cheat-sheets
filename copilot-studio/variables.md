# Copilot Studio : Variables

## Variable Uses

```text
Store user answers
Pass inputs to tools
Receive tool outputs
Control conditions
Preserve conversation context
Build responses
```

## Scopes

```text
Topic scope
Global or agent scope where supported
System-provided values
Tool input and output values
```

Name variables by business meaning, initialize them deliberately, and validate type and presence before use. Avoid storing unnecessary sensitive data, and never treat a client-supplied variable as trusted identity or authorization evidence.
