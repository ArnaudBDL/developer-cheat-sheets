# Markdown : Paragraphs

## Paragraphs

```markdown
This is the first paragraph. It can continue across
multiple source lines and render as one paragraph.

This is the second paragraph.
```

## Line Breaks

```markdown
First line with two trailing spaces.  
Second line.

First line with an HTML break.<br>
Second line.
```

## Thematic Break

```markdown
---

***

___
```

Separate paragraphs with a blank line. Use explicit hard line breaks only when the rendered line boundary matters.
