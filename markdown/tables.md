# Markdown : Tables

## Table

```markdown
| Name  | Role   | Active |
|-------|--------|--------|
| Alice | Admin  | Yes    |
| Bob   | Editor | No     |
```

## Alignment

```markdown
| Left | Center | Right |
|:-----|:------:|------:|
| A    | B      | C     |
```

## Escaped Pipes

```markdown
| Expression | Meaning |
|------------|---------|
| `A \| B`  | A or B  |
```

Tables are a GitHub Flavored Markdown extension rather than a core CommonMark feature. Keep cells concise and prefer lists or headings for complex multi-paragraph content.
