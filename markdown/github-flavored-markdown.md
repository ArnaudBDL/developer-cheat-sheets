# Markdown : GitHub Flavored Markdown

## Task Lists

```markdown
- [x] Define the schema
- [ ] Implement the API
- [ ] Add integration tests
```

## Strikethrough

```markdown
~~Deprecated option~~
```

## Autolinks

```markdown
https://example.com
contact@example.com
```

## Tables

```markdown
| Feature | Status |
|---------|--------|
| API     | Ready  |
```

## Fenced Code

````markdown
```json
{
  "status": "ready"
}
```
````

GitHub Flavored Markdown is a strict superset of CommonMark. Its specified extensions include tables, task list items, strikethrough and extended autolinks. GitHub also performs additional processing and HTML sanitization on user content.
