# Markdown : Quick References

## Text

```markdown
# Heading 1
## Heading 2

**bold**
*italic*
~~strikethrough~~
`inline code`
```

## Links and media

```markdown
[Link label](https://example.com)
![Alternative text](image.png)

> Blockquote
```

## Lists and tasks

```markdown
- Item
  - Nested item

1. First
2. Second

- [ ] Todo
- [x] Done
```

## Code and tables

```markdown
```bash
echo "Hello"
```

| Name | Value |
|------|-------|
| Key  | Data  |
```
