# Markdown : Lists

## Unordered Lists

```markdown
- First item
- Second item
  - Nested item
  - Another nested item
```

## Ordered Lists

```markdown
1. First step
2. Second step
3. Third step
```

## Multiple Blocks in an Item

````markdown
1. First item paragraph.

   Additional paragraph in the same item.

   ```bash
   echo "Code in the item"
   ```

2. Second item.
````

## Rules

- Keep indentation consistent.
- Use ordered lists when sequence matters.
- Add blank lines when list items contain multiple blocks.
- Avoid mixing marker styles without a reason.
