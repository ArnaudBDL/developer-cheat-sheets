# Markdown : Syntax

## Document Structure

````markdown
# Document Title

Introductory paragraph.

## Section

- First item
- Second item

```text
Code block
```
````

## Escaping

```markdown
\*Literal asterisks\*
\# Not a heading
\[Not a link\]
```

## Characters Commonly Escaped

```text
\ ` * _ { } [ ] < > ( ) # + - . ! |
```

Markdown is plain text syntax for structured documents. Rendering details can vary between parsers and dialects, so use the syntax supported by the target platform.
