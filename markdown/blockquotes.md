# Markdown : Blockquotes

## Basic Blockquote

```markdown
> Quoted text.
>
> Second paragraph in the quotation.
```

## Nested Blockquote

```markdown
> First level
>
>> Second level
```

## Other Blocks Inside

```markdown
> ### Quoted heading
>
> - First item
> - Second item
```

Use blockquotes for quoted or attributed material, not merely as a general-purpose visual callout unless the target platform defines that convention.
