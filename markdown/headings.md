# Markdown : Headings

## ATX Headings

```markdown
# Heading level 1
## Heading level 2
### Heading level 3
#### Heading level 4
##### Heading level 5
###### Heading level 6
```

## Setext Headings

```markdown
Heading level 1
===============

Heading level 2
---------------
```

## Rules

- Keep a space between the hash marks and heading text.
- Use heading levels in a logical hierarchy.
- Avoid using bold text as a substitute for document structure.
- Keep headings concise and descriptive.
- Do not rely on automatically generated anchor formats across platforms.
