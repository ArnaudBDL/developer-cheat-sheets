# Prompt Engineering : Quick References

## Core template

```text
Role: relevant expertise
Context: only useful facts
Task: explicit action
Constraints: boundaries and exclusions
Output: required format
Quality criteria: acceptance conditions
```

## Techniques

```text
Few-shot examples
Structured output schemas
Task decomposition
Tool grounding
Retrieval grounding
Self-check against acceptance criteria
```

## Anti-patterns

```text
Vague objectives
Conflicting instructions
Unbounded scope
Missing output format
Unverified assumptions
Examples that contradict the rules
```
