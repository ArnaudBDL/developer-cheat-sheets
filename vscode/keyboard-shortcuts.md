# Visual Studio Code : Keyboard Shortcuts

## Common Shortcuts

```text
Command Palette          Ctrl+Shift+P / Cmd+Shift+P
Quick Open               Ctrl+P / Cmd+P
Settings                 Ctrl+, / Cmd+,
Keyboard Shortcuts       Ctrl+K Ctrl+S / Cmd+K Cmd+S
Integrated Terminal      Ctrl+`
Find                     Ctrl+F / Cmd+F
Workspace Search         Ctrl+Shift+F / Cmd+Shift+F
Run and Debug            Ctrl+Shift+D / Cmd+Shift+D
Extensions               Ctrl+Shift+X / Cmd+Shift+X
Source Control           Ctrl+Shift+G / Cmd+Shift+G
Start Debugging          F5
```

## keybindings.json

```json
[
  {
    "key": "ctrl+alt+t",
    "command": "workbench.action.terminal.new"
  }
]
```

Resolve conflicts in Keyboard Shortcuts, inspect the command ID, and use `when` clauses for context-specific bindings.
