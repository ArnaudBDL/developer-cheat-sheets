# Visual Studio Code : Installation

## CLI Verification

```bash
code --version
code --help
code .
```

## Command-Line Setup

```text
Windows and Linux installers can add code to PATH.
On macOS, run "Shell Command: Install 'code' command in PATH" from the Command Palette.
```

## Common Commands

```bash
code .
code file.txt
code --new-window .
code --reuse-window .
code --diff old.txt new.txt
```

- Install the stable or approved channel for the environment.
- Keep VS Code and extensions updated according to organizational policy.
- Verify platform requirements before installation.
- Use an approved extension marketplace or VSIX process on restricted systems.
