# Visual Studio Code : Git Integration

## Core Workflow

```text
Open or clone repository
Review changes
Stage selected changes
Enter commit message
Commit
Fetch or pull
Push or synchronize
```

## Commands

```text
Git: Clone
Git: Initialize Repository
Git: Stage All Changes
Git: Commit
Git: Fetch
Git: Pull
Git: Push
Git: Create Branch
Git: Checkout to
Git: View History
```

## Configuration

```json
{
  "git.autofetch": true,
  "git.confirmSync": true,
  "git.enableSmartCommit": false
}
```

Review diffs before staging and committing. Resolve merge conflicts carefully, verify the active branch and remote, and keep authentication outside repository settings.
