# Visual Studio Code : Profiles

## Profile Contents

```text
Settings
Keyboard shortcuts
Extensions
User snippets
Tasks
UI state
```

## Typical Profiles

```text
Frontend
Backend
Documentation
Data Science
Remote Development
Minimal Troubleshooting
```

## Commands

```text
Profiles: Create Profile
Profiles: Switch Profile
Profiles: Export Profile
Profiles: Import Profile
```

Use profiles to isolate toolsets by project or workflow. Export only non-sensitive configuration. A minimal empty profile is useful for diagnosing problems caused by settings or extensions.
